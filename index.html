Data Cleaning in SQL

In this project, I did the process of data cleaning or data cleansing on the Nashville housing dataset. This is one of the initial and important steps before performing any data analysis. I standardized the date formats, populated the null Property Address field, segregated the address, city, and state on different columns to make it easier while analysis. I removed the duplicates from the dataset by using Common Table Expressions and Partition By clause. Lastly standardized some fields with different types of data such as ‘Yes’ and ‘Y’ to standard data across the dataset.

Data Exploration in SQL

In this project, I used the Covid-19 records, which contain information on the number of people who died due to the virus and the number of people who were vaccinated against it. After the initial data analysis I gained quite a few insights: The likelihood of dying if the virus is contracted in a specific location, The percentage of people who got affected by covid across the world, Countries and Continents with the highest death rate, The number of vaccinations rolling out on daily basis based on the location. I also used Common Table Expressions to check the percentage of vaccinations done till date. Lastly, I created a View to use it while visualizing.

Data Visualization using Tableau

In this project, I visualized the data gained by analysis in 4 different ways. The death percentage is displayed in a table along with Total cases and deaths. The total deaths per continent is visualized in a bar graph which shows that Europe has the highest total deaths and Oceania has the lowest total deaths. The next visualization is on the world map with the percentage of population infected per country. Lastly, I visualized the percentage of population infected in countries like Canada, China, India, Mexico, United Kingdom, United States and forecasted the population which will be infected by the next 6 months. As per the forecast, UK will be the country with highest infected population.
